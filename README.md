
# Project Title

A brief description of what this project does and who it's for

This is the framework for web automation using playwright with js with POM
## Run Locally

Clone the project

```bash
  git clone https://github.com/RameshQA-17/PlaywrightframeworkwithPOM.git
```

Go to the project directory

```bash
  cd PlaywrightframeworkwithPOM
```

Install dependencies

```bash
  npm install
```

To run the test 

```bash
  npm run test
```
To run the smoke test 

```bash
  npm run smoke
